# Bibliography taxonomy — open-world preservation list

This is a preservation list, not a claim of literal completeness.

The word “bibliography” covers multiple layers that should stay separate:

- conceptual models
- cataloging / description rules
- record schemas
- interchange formats
- metadata schemas
- ontologies / vocabularies
- application profiles
- citation-input formats
- citation-output style rules
- protocols / transport envelopes
- identifier systems
- adjacent resource-description schemas

The inventory below expands the seed list already captured in the handoff. Items are grouped by what they are, not by how useful they are.

## 10) Preserved distinction rules

- IFLA LRM is a conceptual model, not a record format.
- MARC 21 is a family of record formats, not one monolith.
- MODS and MADS are XML metadata schemas.
- CSL-JSON and BibTeX are interchange formats.
- Crossref and DataCite are registration schemas.
- JATS is a publication-content schema with references.
- BIBO / FaBiO / CiTO / BiRO are ontologies, not citation styles.
- ISO 690 / APA / MLA / Chicago are output rules, not storage schemas.
- DOI is an identifier system, not a bibliography schema.
- OAI-PMH is a protocol; it can carry bibliographic payloads but is not itself a bibliographic schema.

## 11) Architecture note

If this taxonomy becomes a registry, keep the layers separate:

- canonical internal record
- source-preserving payloads
- semantic graph
- output formatting
- validation
- crosswalk registry
- authority layer

That prevents schema inflation and makes future expansion auditable.
